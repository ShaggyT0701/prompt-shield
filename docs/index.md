# prompt-shield

Self-learning prompt injection detection engine for LLM applications.

## Overview

prompt-shield detects and blocks prompt injection attacks targeting LLM-powered applications. Unlike static tools, it features a self-hardening feedback loop where every blocked attack strengthens future detection.

## Quick Start

Install: `pip install prompt-shield`
